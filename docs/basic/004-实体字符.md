# 实体字符

用特定的实体编码表示的字符序列，在 `HTML` 文档中显示特殊字符或字符实体。以 `&` 开始，以 `;` 结束。

| 字符         | 实体编码 | 显示 |
| ------------ | -------- | ---- |
| 非断行空格   | `&nbsp;` |      |
| 小于号       | `&lt;`   | <    |
| 大于号       | `&gt;`   | >    |
| 和号         | `&amp;`  | &    |
| 双引号       | `&quot;` | "    |
| 单引号       | `&apos;` | '    |
| 版权符号     | `&copy;` | ©    |
| 注册商标符号 | `&reg;`  | ®    |
